ఈ మాసపు పెద్దలు రాసిన కథల్లో భాగంగా మాతృదేశపు ఔన్నత్యపు సౌందర్యాన్ని దర్శింపజేసే  రష్యన్ కథ  <a href="http://kottapalli.in/2016/02/జనని-జన్మభూమి" target="_blank"> <b>జననీ జన్మభూమి</b></a>ని అందుకోండి.  రాధ మండువ గారి అనుసృజన దీనికి చక్కని రూపాన్నిస్తే, శివ వేసిన చిత్రాలు మరింత గొప్ప సొబగులని అద్దాయి.  వీరికి ఇద్దరికీ అభినందనలు, ధన్యవాదాలు.

సంపదని ఎలా ఉంచుకోవాలో, ఎలా పంచుకోవాలో మానవ సమాజానికి ఇంకా అర్థం కాలేదు.  ఈ మాసపు కొత్త బేతాళం కథ <a href="http://kottapalli.in/2016/02/పీడాంతకం" target="_blank"> <b>పీడాంతకం</b></a> తనదైన నేపథ్యంలో ఈ సమస్యని ప్రస్తావిస్తున్నది.  'అందరం శాంతి సౌఖ్యాలతో బ్రతకాలంటే స్వార్థానికి మందు కనుక్కోక తప్పదు' అని గుర్తుచేసేందుకు చూస్తుంది.  చదివి చెప్పండి మరి.

ఆఫ్రికన్ పిల్లాడు క్వేకుజిన్ గొప్ప సాహసమే చేశాడు. చెప్పిన మాటల్లా విని ఎంత సాధించుకొచ్చాడో చూడండి, <a href="http://kottapalli.in/2016/02/క్రూరజంతువుల_కథ-1" target="_blank"> <b>క్రూరజంతువుల కథ</b></a> - మొదటి భాగంలో. దీని ఆంగ్ల మూలాన్ని అందించిన Berty <a href="http://storynory.com" target="_blank"> <b>స్టోరీ నోరీ డాట్ కాం -storynory dot com</b></a>  వారికి ధన్యవాదాలు.  కథ ముగింపు అయితే, మరి వచ్చే మాసం.

కాకి నలుపుది, పావురం నీలంది.  కాకిని చిన్నచూపు చూసిన పావురం తన తప్పుని ఎలా గుర్తించిందీ, తర్వాత అవి రెండూ కలిసి ఏం చేసిందీ <a href="http://kottapalli.in/2016/02/స్నేహం_విలువ" target="_blank"> <b>స్నేహం విలువ</b></a>  లో చూడండి.   జంతువుల్ని ప్రేమించటం గురించి హిమశ్రీ రాసిన ఓ గందరగోళం <a href="http://kottapalli.in/2016/02/ఆప్యాయత" target="_blank"> <b>ఆప్యాయత</b></a> మరి మీ పిల్లలకి నచ్చుతుందేమో చూడండి.  'తెలివి ఒకరి సొత్తు కాదు. అందరికీ తెలివి ఉంటుంది' అని చిన్నవాడైనా పెద్దలకు కూడా గుర్తు చేస్తున్నాడు అహ్మద్, తన <a href="http://kottapalli.in/2016/02/తెలివి" target="_blank"> <b>తెలివి</b></a> కథలో. చదివితే బలే కోణాలు బయటపడతై. చదవండి మరి.  'గౌరవం మనకా? మనం వేసుకునే బట్టలకా?' ఆలోచిస్తే చురకలాగా తగులుతుంది ఏడవ తరగతి పృథ్వీ గుర్తు చేసిన <a href="http://kottapalli.in/2016/02/వేషం_గొప్ప" target="_blank"> <b>వేషం_గొప్ప</b></a> కథ.  

గురజాడ అప్పారావుగారు పోయి వందేళ్ళు నిండింది.  ఆయన్ని స్మరించుకుంటూ పదోతరగతి జగదీష్ రాసిన పద్య గాధ <a href="http://kottapalli.in/2016/02/గ్రద్ద-పావురం" target="_blank"> <b>గ్రద్ద-పావురం</b></a>.    స్వేచ్ఛగా పారే కవనంలోని మాధుర్యాన్ని మనకు గుర్తుచేసేందుకు వచ్చాయి,   గురజాడవారి <a href="http://kottapalli.in/2016/02/దేశమును_ప్రేమించుమన్నా" target="_blank"> <b>దేశమును_ప్రేమించుమన్నా</b></a> , నండూరి రామకృష్ణమాచార్యులవారి <a href="http://kottapalli.in/2016/02/తప్పు_నాది_కాదు" target="_blank"> <b>తప్పు_నాది_కాదు</b></a> లు రెండూనూ. వీటన్నిటినీ‌ చదివి మీరుకూడా కథల్ని పాటలుగాను, పద్యాలుగాను చెప్పగలరేమో, ప్రయత్నించండి!

"మనం కృషి చేస్తాం, దేవుడు సాయం చేస్తాడు" అని తనదైన శైలిలో, ఆచరణలో అమలు చేసి చూసేందుకు వీలుగా రాశారు, <a href="http://kottapalli.in/2016/02/దేవుని_సాయం" target="_blank"> <b>దేవుని సాయం</b></a> కథను, ఆదూరి హైమవతి గారు.    ఇక వియోగి కలం నుండి వెలువడిన మరో తూటా, <a href="http://kottapalli.in/2016/02/కనువిప్పు" target="_blank"> <b>కనువిప్పు</b></a>. విగ్రహాలకూ, వానలకూ ఏమైనా సంబంధం‌ ఉందని అనుకుంటారా, మీరు?  

గర్వం ఎంత అనవసరమో చెప్పేందుకు సంగీతం మాధ్యమంగా ప్రయత్నించాడు మల్లికార్జున, తన <a href="http://kottapalli.in/2016/02/గర్వభంగం" target="_blank"> <b>గర్వభంగం</b></a> కథలో.  అట్లాగే 'విచారం వద్దు- పరిష్కారాలు వెతకాలి' అంటున్నది నారాయణ <a href="http://kottapalli.in/2016/02/విచారం" target="_blank"> <b>విచారం</b></a>.

ఇన్ని ఉన్నై, ఈ మాసపు కొత్తపల్లిలో. ఇవి కాక, ఈ మాసపు <a href="http://kottapalli.in/2016/02/తెలుసుకుందాం!" target="_blank"> <b>తెలుసుకుందాం!</b></a> చదివి సూడాన్ దేశం గురించి కనుక్కోండి.    దినపత్రికల్లో సూడాన్ గురించి వచ్చే సమాచారాన్ని అర్థం చేసుకునేందుకు దీన్ని ఉపయోగించుకోండి! ఇక పదరంగమూ, పదాల్ని వెతికి పట్టుకోవటమూ అయితే ఉండనే ఉన్నై!
