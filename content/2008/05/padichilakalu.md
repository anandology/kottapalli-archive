పది చిలకలు పాడుతుండగా, <br/>
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;పాడలేక ఒకటిపోతె ఇంకా తొమ్మిది<br/>
తొమ్మిది చిలకలు తూగుచుండగా, <br/>
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;తూగలేక ఒకటిపోతె ఇంకా ఎనిమిది<br/>
ఎనిమిది చిలకలు ఎగురుతుండగా, <br/>
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;ఎగరలేక ఒకటిపోతె ఇంకా ఏడు<br/>
ఏదు చిలకలు ఏడ్చుచుండగా, <br/>
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;ఏడ్చలేక ఒకటిపోతె ఇంకా ఆరు<br/>
ఆరు చిలకలు ఆడుచుండగా, <br/>
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;ఆడలేక ఒకటిపోతె ఇంకా అయిదు<br/>
ఐదు చిలకలు అరచు చుండగా, <br/>
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;అరచలేక ఒకటిపోతె ఇంకా నాలుగు<br/>
నాలుగు చిలకలు నవ్వుతుండగా, <br/>
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;నవ్వలేక ఒకటి పోతె ఇంకా మూడు<br/>
మూడు చిలకలు మూల్గుతుండగా, <br/>
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;మూల్గలేక ఒకటిపోతె ఇంకా రెండు<br/>
రెండు చిలకలు రేగుచుండగా, <br/>
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;రేగలేక ఒకటిపోతె ఇంకా ఒక్కటి<br/>
ఒక్క చిలుక వంచుచుండగా, <br/>
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;వంచలేక ఒకటిపోతె ఇంకా సున్నా.!<br/>
